# Skill: Graft Dev

## Purpose
Implement Graft slices safely with outside‑in development.

## Steps per slice
1. Study the slice requirements in `docs/roadmap/vertical-slices.md`.
2. Add failing tests (CLI subprocess).
3. Implement just enough code to pass.
4. Update docs & schemas.
5. Write notes to `agent-records/work-log/<date>.md`.

## Outputs
- Green tests, updated docs, minimal diffs.
